# student-study-portal
## Overview

The Student Study Portal is a web-based application designed to provide students with a comprehensive platform to manage their academic tasks, access educational resources, and stay organized. Built using Python, Django, HTML, and CSS, this project aims to simplify the learning process and enhance student productivity.

## Features

### 1. Notes
   
   - Create, edit, and delete notes for various subjects and topics
    
   - Organize notes using tags and categories

### 2. Todos
   
   - Create and manage to-do lists for assignments, projects, and study tasks
    
   - Set deadlines and reminders for tasks
    
   - Mark tasks as completed and track progress

### 3. Homework

   - Create and manage homework assignments for various subjects
    
   - Track deadlines and submission status

### 4. YouTube Search

   - Search for educational videos on YouTube
    
   - Filter search results by relevance, view count, and rating

### 5. Dictionary

   - Search for word definitions and meanings

   - Get synonyms, antonyms, and example sentences

   - Learn new words and expand vocabulary

### 6. Wikipedia Search

   - Search for articles and topics on Wikipedia

   - Filter search results by relevance and category

   - Read articles directly within the portal

### 7. Books

   - Search for books by title or subject
 
   - Read book summaries, reviews, and excerpts

### 8. Conversion tools

   - Unit conversion: convert between different units of measurement (e.g., length, weight)
 
### Technical Details
  - Backend: Python 3.x, Django 5.x
    
  - Frontend: HTML5, CSS3, JavaScript
    
  - Database: SQLite (default)
